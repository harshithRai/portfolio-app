# Harshith Portfolio Website (React + Vite)

## Run locally
```bash
npm install
npm run dev
```

## Update your details
Edit `src/data.ts` (email + links + project text).

Put screenshots in `public/images/` and update filenames in `src/data.ts`.

## Build
```bash
npm run build
npm run preview
```

## Deploy
- Netlify/Vercel: build `npm run build`, output `dist`
- GitHub Pages: set `base` in `vite.config.ts` to `/<repo-name>/`
